Shopping Cart

A basic shopping cart system built with HTML, CSS, and JavaScript. Users can add items, adjust quantities, and see real-time total pricing updates. I used JavaScript to handle dynamic cart updates, item removal, and total calculations.

Guessing Game

A simple interactive game where the player tries to guess a randomly generated number between 1 and 20. I used JavaScript and a while loop to keep prompting the user until they guess correctly. The game gives feedback after each guess to guide the player.

Lunch Game

A fun budgeting tool where players manage a $30 weekly lunch allowance. Each day, users enter how many sandwiches they buy, with sandwich prices changing daily. The app tracks spending and shows how much of the weekly budget remains.

Wishlist

A simple and user-friendly app that lets users create and manage a list of items they want. Users can add and view them.At the end users will be able to print their list. Built with HTML, CSS, and JavaScript, the app tracks data dynamically for easy planning.

Pizza App

An interactive app that lets users customize and order their favorite pizzas. Users can choose toppings and quantities with real-time price updates. Built using HTML, CSS, and JavaScript to dynamically update and display the order summary.
